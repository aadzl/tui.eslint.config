# eslint-config-tui

#### ESLint sharable config for TUI components

## Install
```js
npm install eslint-config-tui --save-dev
```

## Usage
Add `.eslintrc` on your project's root directory
```
// .eslintrc
{
    "extends": "tui",
    "rules": {
        // Additional rules
    }
}
```
